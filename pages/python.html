WealthMaker Lead Punch Script Documentation

This script automates the punching of leads into the WealthMaker API (redacted URL due to security concerns).

Functionality

The script performs the following actions:

  • Reads names and mobile numbers from separate text files (`names.txt` and `mobile_numbers.txt`).
  • Ensures both files contain the same number of lines.
  • Iterates through each name and mobile number pair.
  • Constructs a data dictionary with lead information.
  • Sends a POST request to the WealthMaker API with the lead data.
  • Prints the status code and response for each lead.

Important Notes

**Security:**

  • **DO NOT** commit the WealthMaker API URL or token to a public repository. Store them securely using environment variables or a secrets management system.
  • This documentation redacts the URL for demonstration purposes.

Usage

**Prerequisites:**

  • Python 3.x (https://www.python.org/downloads/)
  • Requests library (`pip install requests`)

**Instructions:**

  1. Create two text files named `names.txt` and `mobile_numbers.txt` containing names and mobile numbers (one per line, corresponding to each other).
  2. Update the environment variables with your WealthMaker API credentials (**DO NOT** commit them to the repository).
  3. Run the script from your terminal: `python wealth_maker_lead_punch.py` (replace with the actual script filename).

Technical Details (Optional)

The script utilizes the `requests` library to make HTTP requests to the WealthMaker API. The script iterates through each name and mobile number pair, constructing a data dictionary with various lead information. The dictionary is then sent as JSON data in a POST request to the API.

Random Indian Mobile Number Generator

Purpose

This Python script generates random 10-digit Indian mobile numbers using a specified prefix.

Usage

  1. Ensure you have Python installed on your system.
  2. Save the script as a `.py` file (e.g., `mobile_number_generator.py`).
  3. Run the script from your terminal or command prompt.

Functionality

The script:

  • Defines a function `generate_random_10_digit_number` that takes a 3-digit prefix as input.
  • Generates a random 7-digit suffix.
  • Combines the prefix and suffix to create a 10-digit number.
  • Uses a list of allowed prefixes for Indian mobile numbers.
  • Generates 200 random mobile numbers using the specified prefixes.

Example Output

The script will output 200 random 10-digit Indian mobile numbers, starting with the specified prefixes.

Customization

You can customize the script by:

  • Changing the number of random numbers to be generated.
  • Adding or removing prefixes from the allowed list.
  • Modifying the function to include additional constraints or formatting options.

Note

This script is for demonstration purposes only. It does not guarantee the validity or availability of the generated mobile numbers.
