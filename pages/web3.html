Let me show you my Simple Grading module, which I created for learning purposes in Odoo 17.

Setup

$ sudo apt update

$ cd ~ && mkdir test_module

$ cd test_module && git clone "https://github.com/sadsatan6666/Simple-Module"

$ # assuming you already have odoo installed in your system and you are at at '~/test_module'

$ sudo cp -r Simple-Module /usr/lib/python3/dist-packages/odoo/addons/

$ sudo service postgresql restart && sudo service odoo restart

$ #goto the 'localhost:8069'

$ #now goto the 'Apps' and search 'Simple-Module'

$ #Click on The 'Activate' button

$ #And... Bingo... your module is installed it should be seen by clicking on top left corner of odoo interface

Before

This is a simple dummy 'Student Grading' module that I made while I was (and still am) learning Odoo.

Before this, I made a simpler module called 'Simple Contact Book'. This module had only two fields:

  • Contact Name
  • Contact Number

Like others, you can add, edit, and delete records.

I then made changes to that module, which led to the creation of 'Student Grading'.
